# Text Similarity — Сравнение схожести текстов

Скрипт вычисляет **косинусное сходство** между тремя предложениями
с помощью библиотеки `sentence-transformers` и определяет наиболее похожую пару.

---

## Быстрый старт

### 1. Клонируй / скачай проект

```bash
git clone <repo-url>
cd text-similarity
```

### 2. Создай и активируй виртуальное окружение

```bash
# macOS / Linux
python3 -m venv .venv
source .venv/bin/activate

# Windows
python -m venv .venv
.venv\Scripts\activate
```

### 3. Установи зависимости

```bash
pip install -r requirements.txt
```

> При первом запуске модель (~500 МБ) скачается автоматически с Hugging Face.

### 4. Запусти скрипт

```bash
python main.py
```

## Как изменить предложения

Открой `main.py` и отредактируй список `SENTENCES`:

```python
SENTENCES = [
    "Первое предложение.",
    "Второе предложение.",
    "Третье предложение.",
]
```
